# TL AI Keyframe Workbench

一个面向 AI 视频创作的关键帧图片生成工作台原型。

这个项目目前聚焦在最小 MVP：先把图片关键帧、项目剧本、资产库和帧详情的交互跑通，再接入图片生成模型和视频生成模型。

## 当前能力

- 项目列表与项目设置
- 剧本文本编辑
- 项目资产库
- 资产新建、编辑、替换图片
- 图片关键帧时间轴
- 帧插入、删除、版本切换
- 帧详情记录：概要、时长、人物、对白、动作、情绪、备注
- 类即梦的极简提示词输入舱
- `@` 调用资产和项目帧

## 使用方式

这是一个纯 HTML 原型，可以直接打开：

```bash
open index.html
```

也可以用本地静态服务运行：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://localhost:4173/index.html
```
